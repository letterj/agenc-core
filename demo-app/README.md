# agenc-demo

Internal demo UI for AgenC privacy and protocol flows.

This workspace is the demo-facing web app under `src/` with static assets in
`public/`.

Local commands:

```bash
npm --prefix demo-app run dev
npm --prefix demo-app run build
npm --prefix demo-app run typecheck
npm --prefix demo-app run preview
```

Use this surface for demo-oriented product flows, not as a public SDK example.
