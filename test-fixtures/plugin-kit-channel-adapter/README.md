# Plugin Kit Channel Fixture

Internal fixture package used to exercise the plugin-kit channel-adapter
contract from `agenc-core`.

This fixture is not a public starter template. It exists to support tests and
pack-smoke-style validation inside the core workspace.

Local commands:

```bash
npm --prefix test-fixtures/plugin-kit-channel-adapter run build
npm --prefix test-fixtures/plugin-kit-channel-adapter run typecheck
```
